THUMBNAIL DESIGN SYSTEM Style Index A catalog of six thumbnail templates — each with its own design guide This document is the entry point to the thumbnail design system. Each of the six styles below has its own full template guide in a separate .docx file, covering layout, typography, color, build steps, do/don&apos;t rules, and a pre-publish checklist. A single rule holds across all six: the designer executes the treatment; the copywriter provides the words. Never invent, edit, or abbreviate supplied copy — render it exactly as written. The six styles at a glance # Style name Visual DNA Best for 01 Cinematic B&amp;W Banner Full-bleed B&amp;W photograph, deep-red condensed headline cutting across the midline Documentary features, investigative deep-dives, historical retrospectives 02 Six-Panel Mosaic 2×3 grid of portraits, three hand-drawn chalky labels scattered across selected cells Group showcases, guess-style reveals, ensemble features, quiz content 03 Dual-Flag Backdrop Two flags meeting on a diagonal seam, B&amp;W subject cutout in front, cream italic banner Cross-cultural content, comparison narratives, &apos;X meets Y&apos; stories 04 Faux Social Post Card Dark textured backdrop, white tweet-style card with avatar + handle, color subject portrait Breaking-news reveals, shocking exposes, topical takes with social credibility 05 Editorial Magazine Cover Dark-to-red radial backdrop, oversized outlined serif headline, color portrait, script subtitle Artist retrospectives, tributes, single-subject cinematic features 06 Oversized Headline Card Dark backdrop, massive white card, two-color stacked headline, rough brushstroke accent underline Ranked lists, &apos;greatest of&apos; content, superlative reveals, battle-of-the-best topics Files in this system All template guides are standalone .docx files. Export as JPG under 2 MB. Bottom 60 px stays clear of critical subject detail so YouTube&apos;s timestamp overlay doesn&apos;t cover a face. Test every thumbnail at 240 × 135 px (approximate feed size) before exporting. If it fails there, it fails. Render supplied copy exactly. Never edit, shorten, reorder, or add words. If copy doesn&apos;t fit, go back to the copywriter. Each style has a distinctive typographic signature. Don&apos;t swap typography between styles — that&apos;s what makes each one recognizable. Match the style to the content brief. The table above maps each style to its best use case. If the content doesn&apos;t fit any of the six, propose adding a new style rather than stretching an existing one. How to choose a style Ask these questions in order: Is the content about ONE person? → Style 01 (documentary) or Style 05 (editorial tribute). Is it about a SMALL GROUP (2–4)? → Style 01, 03, or 06, depending on mood. Is it about a LARGE GROUP (6+ subjects)? → Style 02 (mosaic) is the only option. Is there a CROSS-CULTURAL or comparison angle? → Style 03 (dual flag). Is it a RANKED LIST or &apos;greatest of&apos;? → Style 06. Is it a BREAKING / SHOCKING reveal? → Style 04 (social post). Is it a REFLECTIVE TRIBUTE or memorial? → Style 05 (editorial). Six styles. One system. Consistency in the rules, variety in the execution.
